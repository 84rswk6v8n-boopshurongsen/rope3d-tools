# Rope3D 组件

这是一个基于粒子/约束求解的 Cocos Creator 3D 绳索组件。运行时代码放在 `assets/rope3d/scripts`，示例场景和示例控制脚本保留在 `assets/rope3d_demo`。

## 组件

- `Rope3D/3D 绳子`：绳索模拟组件，负责长度约束、碰撞、自碰撞、摩擦和刚体冲量。
- `Rope3D/绳子管状渲染`：绳子的管状网格渲染组件，会被 `Rope3D` 自动要求挂载。

## 当前版本重点

- 支持动态长度、连续碰撞、自碰撞、静摩擦、动摩擦和刚体冲量。
- `Grass Interaction` 是可选扩展能力：如果场景里存在带 `sampleReaction(...)` 接口的草地组件，绳子可以受到草地支撑、阻力和恢复推力。
- Rope3D 不再硬依赖 GrassSystem。单独安装绳子插件时，即使项目里没有草地系统也可以正常编译和运行。

## 基本使用

1. 新建一个节点作为绳子节点，添加 `Rope3D/3D 绳子` 组件。
2. 给绳子节点添加或保留 `MeshRenderer`，`RopeTubeRenderer` 会负责生成管状网格。
3. 设置 `起点锚点` 和 `终点目标`。
4. 设置 `碰撞体根节点`，组件会从该节点及子节点收集 `BoxCollider`。
5. 调整 `绳子半径`、`粒子间距`、`绳子长度`、摩擦和碰撞迭代参数。
6. 如果需要草地反作用，设置 `Grass Root` 或在场景中放置支持 `sampleReaction(...)` 的草地组件。

Inspector 中的公开属性都带有中文显示名和悬浮说明；代码里的英文字段名保持稳定，方便脚本继续调用。
